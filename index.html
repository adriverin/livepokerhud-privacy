Hand logging workflow

Track live poker hands

The purpose of Live Poker HUD is to make recording poker hands in live games quick and effortless.

  • Button position advances automatically after each saved hand.
  • BTN -> can manually advance the button if a hand was missed.
  • Tapping any seat/player pill sets the button directly to that player.
  • Straddle mode can be toggled before preflop action starts, assigns STR to the seat after the big blind, and makes that player act last preflop.
  • All-in button marks the next non-fold action as all-in.
  • Showdown winner selection is required when the hand reaches showdown.
  • Board and player hole-card logging is optional.
  • Hand notes can be added during logging.
  • Save stores the hand locally and updates player HUD stats.
  • Player rows show quick stats: hands, VPIP, PFR, and 3Bet.
